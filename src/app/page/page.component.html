About me

     Hello, my name is Dejan Subotic and I’m a frontend developer. I’m from Bosnia and Herzegovina and my current residence is Graz, Austria. I graduated in economics, department of logistics, but my big desire was to try my hand at programming. I have always thought that programming languages ​​are the languages ​​of the future, especially at the time of the pandemic, when it turned out to be extremely important to optimize the way we work from home, where I found space and time to devote more time to learning and building my future profession. My journey lasts almost a year and a half, where I started to learn on my own what has always interested me. Through this learning process I worked with HTML, CSS, SCSS, SASS, JavaScript, Bootstrap, Angular, NgRX, RxJS, Docker, Google Firebase Firestore, Angular Material, Git, GitHub and now I'm in the process of learning backend: Node.js, Express.js, MongoDB and Mongoose. My sources were online platforms with courses, hundreds of tutorials, many hours of learning and daily research in this field. For me programming is fun and challenging, therefore I'm looking forward for further improvement and learning a new programming languages. The product of my work so far is this website which is conceived as a short portfolio as a way I can present myself best. Below you can see some of my projects and you can download my CV here.

My Work

Online Shop App

This web application was made based on the look of the Nike online shop and was originally designed only for the purpose of exercises for HTML and CSS, but in the process I decided to rework it in Angular and to add certain functionalities and additionally insert some of mine ideas related to page design. For backend data I used Firebase, and the application is completely responsive and customized for all types of devices, tablets, mobile phones and desktops. The site is also hosted and you can visit site by clicking on this link. Also the code is on my GitHub account and if you want to see it you can click on this link.

Tennis App

Unlike the first application, which was basically a copy of the already existing Nike online shop, this time I wanted to make something more personalized using the knowledge gained so far, and something that was exclusively my idea. It is a Tennis application, which I am a huge fan of and an active player too. The application contains information about the players I train with, a gallery of pictures, locations where tournaments are held and etc. The application will be expanded, with new functionalities, such as current player rankings, statistics, new albums, events. This application is more complex than the first application, it uses NgRX to manipulate data, custom components, 3-language localization, authentication, roles, guards, interceptors, it is also hosted on the web and it is responsive to all types of devices. Like the previous one, this application uses the Firebase as a backend. The application contains an Admin role that has the ability to make some changes like delete, add or edit player and photos uploads.
